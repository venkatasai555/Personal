Email Spam Classifier

Project Description

  • Category: AI-ML
  • Technologies used: Python, scikit-learn, NLP techniques
  • GitHub Code: Link

Developed an Email Spam Classifier Project using Python. This project was pivotal in addressing the common challenge of spam mail detection in natural language processing (NLP). Leveraging the scikit-learn library, we implemented a range of machine learning algorithms and tools tailored for text classification tasks. The project involved working with the 'Mail_CSV' dataset, where messages were classified into 'SPAM' or 'NOT' categories. Through meticulous steps, including data loading, feature-label splitting, and numerical conversion using TfidVectorizer, we trained the spam classifier model using the Multinomial Naive Bayes algorithm. The evaluation phase, which included predicting labels for the testing set and assessing performance through accuracy scores and confusion matrices, highlighted the project's success in achieving robust spam detection. This experience significantly enhanced my proficiency in NLP and machine learning applications, leaving me well-equipped for future endeavors in the field.
